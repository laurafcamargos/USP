INSTRUÇÕES INICIAIS:

 1. BAIXE E DESCOMPACTE A PASTA

 2. PELO TERMINAL DO SEU COMPUTADOR, ENCONTRE ONDE ESTÁ LOCALIZADA A PASTA QUE VOCÊ ACABOU DE DESCOMPACTAR E ENTRE NETSA PASTA

 3. FEITO ISSO, BASTA UTILIZAR OS COMANDOS ABAIXO, PELO TERMINAL, PARA QUE O CÓDIGO SEJA COMPILADO E EXECUTADO

COMANDOS PARA COMPILAR E EXECUTAR:

 1. PARA COMPILAR O CÓDIGO FEITO DIGITE: 
	make all
 2. PARA EXECUTAR O CÓDIGO FEITO DIGITE: 
	make run

AO EXECUTAR O CÓDIGO, VOCÊ TERÁ QUE DIGITAR QUAL OPERAÇÃO QUER FAZER NO GRAFO, AS OPERAÇÕES ESTÃO LISTADAS ABAIXO, MAS ASSIM QUE EXECUTAR O CÓGITO VOCÊ TAMBÉM VERÁ NA SUA TELA

OPERAÇÕES:

 Digite: 
 	1 Para criar um Grafo
 	2 Para verificar a existência de uma aresta
 	3 Para obter os vértices adjacentes a outro
 	4 Para remover uma aresta
 	5 Para imprimir os vértices e arestas
 	6 Para obter o número de vértices do grafo
 	7 Para excluir o grafo e sair do programa
 	8 Para inserir uma aresta
 	9 Para remover a aresta de menor peso
 	10 Para obter a matriz de adjecência

 OBS.: APÓS INSERIR O NUMERO DA OPERAÇÃO QUE DESEJA, BASTA CLICAR NO ENTER QUE AS PROXIMAS INSTRUÇÕES OU A SAÍDA IRÁ APARECER NA SUA TELA

ALGUMAS EXPLICAÇÕES SOBRE O FUNCIONAMENTO DO CÓDIGO:

 UTILIZAMOS -1 PARA INDICAR ARESTA NÃO EXISTENTE

 NOSSA MATRIZ DE ADJACENCIA IMPRIME -1 PARA ARESTAS NAO EXISTENTES E O PESO INSERIDO DA RESPECTIVA ARESTA SE ELA EXISTENTE.
